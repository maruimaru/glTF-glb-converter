# glTF-glb-converter
Developed to bridge the gap between RealityScan and Nomad Sculpt. This is a fully client-side web application that converts glTF folders into a single GLB file without uploading any data.


# glTF → GLB コンバーター

ブラウザだけで動く **glTF → GLB 変換ツール**です。  
フォルダを選択するだけで、Nomad Sculpt などのアプリで開ける `.glb` ファイルを生成できます。

## このツールで何ができるか

3Dモデルのファイルには様々な形式があります。  
Blender や Sketchfab からエクスポートすると、よく以下のような「バラバラなファイル群」が生成されます。

```
my_model/
├── scene.gltf        ← モデルの構造を記述したテキストファイル
├── scene.bin         ← 頂点・アニメーションなどの生データ
└── textures/
    └── material_baseColor.jpeg  ← テクスチャ画像
```

このツールは、これらをまとめて **1つの `.glb` ファイル** に変換します。

```
scene.glb  ← すべてが1ファイルに！
```

`.glb` にすることで、**Nomad Sculpt・Reality Composer・各種 3D ビューアー** などにそのまま読み込めるようになります。

## 使い方

### 1. ツールを開く

`index.html` をダブルクリックしてブラウザで開きます（インターネット接続不要）。

### 2. フォルダを選択する

「📂 フォルダをクリックして選択」をクリックし、  
`.gltf` ファイルが入っているフォルダを選びます。

> **ドラッグ＆ドロップも可能**  
> フォルダごと、またはファイルを複数まとめてブラウザ画面にドロップしても動作します。

### 3. ファイルを確認する

選択後、検出されたファイルの一覧が表示されます。

| バッジ | 説明 |
|---|---|
| `GLTF` | モデル構造ファイル（必須） |
| `BIN` | バイナリデータ（頂点情報など） |
| `IMG` | テクスチャ画像（JPG / PNG など） |

### 4. 変換する

「**変換して .glb をダウンロード**」ボタンを押すと変換が始まります。  
完了すると「⬇ ファイル名.glb をダウンロード」ボタンが現れるので、クリックして保存してください。

## 対応ファイル構成

| ファイル | 必須 | 備考 |
|---|---|---|
| `.gltf` | ✅ | 1つだけ必要。出力ファイル名の元になります |
| `.bin` | 推奨 | ジオメトリ（形状）データ |
| `.jpg` / `.jpeg` | 任意 | テクスチャ画像 |
| `.png` | 任意 | テクスチャ画像 |
| `.webp` | 任意 | テクスチャ画像 |

複数テクスチャが含まれるモデルにも対応しています。

## 技術的な背景（初心者向け解説）

### glTF とは？

**glTF（GL Transmission Format）** は、3Dモデルを効率的に転送・表示するためのファイル形式です。  
「3Dの JPEG」とも呼ばれ、Web や AR/VR の分野で広く使われています。  
Khronos Group（OpenGL を作った団体）が標準化しています。

### `.gltf` と `.glb` の違い

| 形式 | 構造 | 特徴 |
|---|---|---|
| `.gltf` | テキスト (JSON) + 外部ファイル | 人間が読めるが、複数ファイルに分かれる |
| `.glb` | バイナリ（1ファイル） | 配布・転送に便利。アプリへの読み込みが簡単 |

`.gltf` はメモ帳で開くと中身が読めるテキスト形式です。  
`.glb` はそのテキスト情報とバイナリデータを1つのファイルにパッキングしたものです。

### GLB ファイルの内部構造

GLB ファイルは以下の「チャンク（塊）」で構成されています。

```
[ ヘッダー 12バイト ]
  ├─ マジックナンバー "glTF"
  ├─ バージョン番号 (= 2)
  └─ ファイル全体のサイズ

[ JSON チャンク ]
  ├─ チャンクサイズ
  ├─ チャンク種別 "JSON"
  └─ gltf の JSON テキスト（4バイト境界にスペースでパディング）

[ BIN チャンク ]
  ├─ チャンクサイズ
  ├─ チャンク種別 "BIN\0"
  └─ バイナリデータ（頂点・テクスチャ画像などを連結）
```

このツールは以下の処理を行います。

1. `.gltf` の JSON を読み込む
2. `.bin` ファイルをバイナリとして読み込む
3. 各テクスチャ画像をバイナリとして読み込み、`.bin` の後ろに連結する
4. JSON 内の `images[].uri`（外部ファイルパス）を `bufferView` への参照に書き換える
5. すべてを GLB 形式のバイナリに組み立て、ダウンロードさせる

### なぜブラウザだけで動くのか？

このツールは **JavaScript の FileReader API** を使って、ローカルファイルを直接読み込んでいます。  
サーバーへのアップロードは一切行わず、すべての処理がブラウザ内（あなたのPC上）で完結します。

## よくある質問

**Q. ファイルがサーバーにアップロードされますか？**  
A. されません。すべての処理はブラウザ内で完結しています。大容量ファイルでも安心して使えます。

**Q. テクスチャが複数あっても大丈夫ですか？**  
A. 対応しています。フォルダ内のすべての画像ファイルを自動的に検出して埋め込みます。

**Q. `.bin` ファイルがない場合は？**  
A. `.bin` なしの glTF（データが JSON 内に Base64 で埋め込まれているもの）にも対応しています。

**Q. 変換後のファイルが Nomad Sculpt で開けません**  
A. `.glb` ファイルをiPad/Androidに転送する際は、AirDrop・Google Drive・iCloud Drive などを経由してください。転送後、Nomad Sculpt の「Import」から読み込みます。
